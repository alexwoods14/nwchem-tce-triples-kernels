nwchem-tce-triples-kernels
==========================

NWChem TCE CCSD(T) loop-driven kernels for performance optimization experiments

Set TARGET to one of the following:

CRAY-INTEL
CRAY-GCC
CRAY (Cray compilers)
TUKEY-INTEL
TUKEY-GCC
KAYLA (GCC on ARM)
BGQ (BGXLC)
POWER7-AIX
POWER7 (XLC)
POWER7-GCC
MAC-CLANG
MAC (GCC)
INTEL
MIC (Intel)
